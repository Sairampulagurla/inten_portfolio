# 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website built using **HTML5** and **CSS3**. This project showcases my profile, skills, and contact information with a clean, user-friendly interface.

## 📌 Features

- 🏠 Home page with introduction
- 👨‍💻 About Me page
- 📩 Contact form
- 🌙 Dark Mode toggle
- 📱 Responsive design for Mobile, Tablet, and Desktop
- 🎨 Modern UI using CSS Variables
- ♿ Semantic HTML for better accessibility

- JavaScript (Dark Mode Toggle)
